# ETHBerlin4 Website

Join the ETHBerlin Matrix space to contribute: [#ethberlin:dod.ngo](https://matrix.to/#/%23ethberlin:dod.ngo)

# Run

The website is built with Gatsby

Install dependencies

```
npm install
```

use npx to run without the global dependency

```
npx gatsby develop
```

# Build

Build with

```
npm run build
```
